# Tempo Explorer

## Getting Started

To run this application:

```bash
pnpm install
pnpm dev
```

### Styling

This project uses [Tailwind CSS](https://tailwindcss.com/) for styling.

### Linting, Formatting & Type Checking

This project uses [Biome](https://biomejs.dev/) for linting and formatting.

To format & lint:

```bash
pnpm check
```

To check types:

```bash
pnpm check:types
```
